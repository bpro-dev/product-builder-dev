# 로또 번호 추첨 생성기 (Lotto Number Generator) - Blueprint

## 프로젝트 개요
사용자가 버튼을 클릭하여 1부터 45까지의 숫자 중 중복되지 않는 6개의 숫자를 무작위로 추첨하는 현대적이고 직관적인 웹 애플리케이션입니다.

## 주요 기능 및 디자인 (Baseline Features)
- **추첨 애니메이션**: 번호가 순차적으로 나타나며 긴장감을 조성하는 애니메이션 효과.
- **Web Components**: `LottoBall` 컴포넌트를 사용하여 독립적인 스타일과 로직 관리.
- **현대적 CSS**: `oklch` 색상 모델, `backdrop-filter` (유리 질감), `:has()` 선택자, 반응형 디자인.
- **번호 이력**: 이전 추첨 결과들을 하단에 목록으로 표시.
