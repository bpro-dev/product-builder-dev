# 프로젝트 개요: 행운의 로또 번호 추첨기

이 프로젝트는 사용자가 클릭 한 번으로 무작위 로또 번호를 생성할 수 있는 현대적이고 아름다운 웹 애플리케이션입니다.

## 주요 기능 및 디자인

- **로또 번호 생성**: 1부터 45 사이의 무작위 숫자 6개를 생성합니다.
- **테마 전환**: 라이트 모드와 다크 모드를 지원하여 사용자 경험을 향상시킵니다.
- **최근 이력**: 최근 생성된 번호 세트를 하단에 리스트 형태로 표시합니다.
- **디자인 스타일**: 
  - Glassmorphism: 투명도와 블러 효과를 활용한 현대적인 카드 디자인.
  - OKLCH 색상: 최신 브라우저 표준을 활용한 생동감 넘치는 색상 체계.
  - 반응형 디자인: 모바일과 데스크톱 모두에서 최적화된 레이아웃.

## 최신 업데이트: 제휴 문의 폼 (Formspree)

사용자가 제휴 관련 문의를 보낼 수 있는 간단한 폼을 추가합니다.

### 구현 계획
1. **HTML 구조**: 
   - `index.html` 하단에 `<section class="contact-area">` 추가.
   - Formspree 엔드포인트(`https://formspree.io/f/mgonyqav`)를 사용하는 `<form>` 구현.
   - 성함, 이메일, 문의 내용 입력 필드 포함.
2. **CSS 디자인**:
   - 기존의 Glassmorphism 스타일과 일치하는 입력 필드 스타일링.
   - 다크 모드/라이트 모드 자동 대응.
   - 시각적으로 깔끔하고 직관적인 레이아웃.
3. **기능**:
   - Formspree를 통한 안정적인 이메일 전송.
   - 필수 입력값 검증(HTML5 `required` 속성 활용).
